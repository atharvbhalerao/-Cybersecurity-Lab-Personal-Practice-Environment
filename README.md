# 🔐 Cybersecurity Lab — Personal Practice Environment

## 👨‍💻 About
Self-built cybersecurity lab environment for hands-on practice in 
penetration testing, VAPT, SOC fundamentals, and network security.
All projects performed in isolated virtual environments using 
Oracle VirtualBox, Kali Linux, and Metasploitable 2.

## 🖥️ Lab Setup
- Attacker Machine: Kali Linux (Oracle VirtualBox)
- Target Machine: Metasploitable 2
- Android Emulator: MuMu Player
- Network: Isolated host-only adapter

## 🛠️ Tools Used
Nmap · Metasploit · BurpSuite · Wireshark · Aircrack-ng · 
MSFvenom · OWASP ZAP · Ettercap · Kali Linux · Netcat

## 📁 Projects

### 1. Metasploitable 2 — Full Exploitation Lab
- Performed SSH bypass, FTP anonymous login exploitation
- SMB enumeration and brute forcing using Metasploit
- Port 1524 shell access
- Service version detection and CVE mapping using Nmap

### 2. Web Application Attack Simulation
- SQL Injection on DVWA and testphp.vulnweb.com
- XSS — Reflected and Stored attacks
- Session Hijacking via cookie theft
- Firewall bypass using BurpSuite request interception
- Documented findings per VAPT reporting standards

### 3. Android Hacking — Simulated Environment
- Generated Android reverse shell payload using MSFvenom
- Deployed payload on MuMu Android emulator
- Established Meterpreter session for remote access
- Performed in fully isolated environment

### 4. Wi-Fi Attack Simulation
- Wi-Fi deauthentication attack using Aircrack-ng
- WPA2 handshake capture on personal test network
- Packet injection and traffic analysis using Wireshark

### 5. Network Traffic Analysis — Packet Sniffing
- Captured live network traffic using Wireshark
- Analyzed ARP poisoning patterns
- Identified DNS queries and HTTP cleartext credentials
- Traced TCP 3-way handshake across OSI layers

### 6. Malware Creation & Threat Analysis
- Created reverse shell and bind shell payloads using MSFvenom
- Studied malware behavior and persistence mechanisms
- Analyzed detection evasion techniques
- All testing done in isolated VM environment only
